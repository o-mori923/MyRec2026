# MyRec 2026

作業記録アプリ **MyRec V0.16**（2007年製 MFC アプリ）を Python / tkinter で再実装したものです。  
一定間隔でポップアップし、現在の作業内容を記録するシンプルなタイムトラッキングツールです。

## 機能

| 機能 | 説明 |
|---|---|
| 定期ポップアップ | 設定した間隔（デフォルト15分）で自動的にウィンドウが前面に出る |
| 作業内容の記録 | `[HH:MM-HH:MM]:作業内容` 形式で日付別テキストファイルに追記 |
| **履歴プルダウン** | 過去に入力した作業内容をコンボボックスから選択可能（使用回数順） |
| **日毎の集計** | 「集計」ボタンで年月選択ダイアログを表示。指定月の作業時間を CSV に出力（横持ち・時間単位） |
| ファイルを開く | 「開く」ボタンで今日の記録ファイルをテキストエディタで表示 |

## 使い方

### 起動

```
MyRec2026.exe
```

### 基本操作

1. 起動するとウィンドウがタスクバーに最小化された状態で常駐します
2. 起動直後から手動で作業内容を入力して **[記録]** を押せます
3. 設定した間隔の境界時刻（毎時 00/15/30/45 分など）になると自動でウィンドウが前面に出ます
4. コンボボックスに作業内容を入力（または履歴から選択）して **[記録]** をクリック
5. ウィンドウが再びタスクバーに戻ります

### ボタン一覧

| ボタン | 動作 |
|---|---|
| 記録 | 表示中の時間帯と作業内容を記録し、最小化する |
| 開く | 今日の記録ファイルをテキストエディタで開く |
| 集計 | 年月選択ダイアログを開く。指定年月の作業時間を `YYYY_MM_summary.csv` に出力して開く |

### 終了

**メニュー → 終了**  
または ウィンドウの [X] ボタン（確認ダイアログが出ます）

## ファイル構成

```
MyRec2026.exe        ← アプリ本体
config.txt           ← 設定ファイル（起動時に自動生成）
history.csv          ← 作業内容履歴（自動生成・更新）
YYYYMMDD.txt         ← 日別記録ファイル（Shift-JIS）
YYYY_MM_summary.csv  ← 月次集計 CSV（集計ボタンで生成）
```

## 設定（config.txt）

```
//記録間隔(分単位)
15
//ビープ音(0:OFF/1:ON)
1
//入力なし時の記録(0:しない/1:する)
1
//時計のチェック間隔(秒単位)
10
//ポップアップ時非アクティブ(0:アクティブ/1:非アクティブ)
1
```

| 設定項目 | 説明 | 推奨値 |
|---|---|---|
| 記録間隔 | ポップアップ間隔（分）。0 は 10 扱い | 5 / 10 / 15 / 20 / 30 |
| ビープ音 | ポップアップ時に Windows の通知音を鳴らすか | 1 |
| 入力なし時の記録 | 未入力のまま次の間隔を迎えた場合、空エントリを自動追記するか | 1 |
| 時計チェック間隔 | 内部タイマーの精度（秒）。小さくするほど CPU 負荷が上がる | 10 |
| ポップアップ時非アクティブ | 1 にすると他のウィンドウのフォーカスを奪わずにポップアップ | 1 |

> **動作テスト時のヒント：** `記録間隔` を `1` にすると毎分ポップアップするので動作確認に便利です。確認後は `15` に戻してください。

## 記録ファイルの形式

```
■2026-06-10
[09:00-09:15]:メール確認
[09:15-09:30]:設計作業
[09:30-09:45]:
```

- 先頭行: `■YYYY-MM-DD`
- 以降: `[開始時刻-終了時刻]:作業内容`
- 空エントリ（`[HH:MM-HH:MM]:` のみ）は未入力の時間帯を表す

## 集計 CSV の形式

```
日付,設計作業,メール確認,コードレビュー,...
2026-06-09,2.5,0.25,1.0,...
2026-06-10,1.5,0.5,0.0,...
```

- 1 行 = 1 日
- 列 = 作業内容（当月に登場した全作業内容）
- 値 = 合計時間（時間単位、小数第2位まで）

## 開発・ビルド

```powershell
# 依存関係インストール
uv sync

# スクリプトとして実行
uv run python myrec.py

# exe ビルド
uv run pyinstaller --onefile --windowed --name MyRec2026 --icon myrec.ico --add-data "myrec.ico;." myrec.py
# → dist\MyRec2026.exe に出力される
```
